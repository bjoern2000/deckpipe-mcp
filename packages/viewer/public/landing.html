deckpipe

Agent-first slide deck engine. Describe slides as JSON. Get rendered decks.

Deckpipe lets AI agents create, edit, and export presentation decks through a simple API. No GUI needed. Agents describe slides using structured JSON with layouts, content, and themes — Deckpipe handles rendering, styling, and PDF export.

Built for the agentic workflow: your AI assistant creates the deck, you review it in the browser, edit inline, and export.

Install the MCP Server

Add Deckpipe to Claude Code, Cursor, or any MCP-compatible client:

# Claude Code
claude mcp add deckpipe -- npx @deckpipe/mcp-server

Or add to your MCP config manually:

{
  "mcpServers": {
    "deckpipe": {
      "command": "npx",
      "args": ["-y", "@deckpipe/mcp-server"]
    }
  }
}

How It Works

  1. Install the MCP server in your AI coding tool
  2. Ask your agent to create a slide deck on any topic
  3. The agent calls create_deck with structured slide data
  4. Open the returned viewer URL to see your rendered deck
  5. Edit slides inline in the browser, or ask the agent to update them
