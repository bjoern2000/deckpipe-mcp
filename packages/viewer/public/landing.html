deckpipe

agent-first slide decks. describe slides as JSON, get rendered decks.

AI agents create and edit decks through a simple API. Structured JSON in, rendered slides out. Review in the browser, edit inline.

Install

# Claude Code
claude mcp add deckpipe -- npx deckpipe-mcp

Or add manually:

{
  "mcpServers": {
    "deckpipe": {
      "command": "npx",
      "args": ["-y", "deckpipe-mcp"]
    }
  }
}

Tools

create_deck get_deck update_deck delete_deck upload_image list_layouts
